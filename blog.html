What are the different ways to select an element in the DOM?
There are most important DOM selector getElementById,getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll
What is the difference between innerHTML, innerText, and textContent ?
innerHTML reads the HTML markup and the text content of the element, innerText returns the text as it appears on screen. It ignores HTML tags and the textContent reads text as it is in the markup. It also returns all text.
What is event delegation in the DOM?
Event delegation is a design style in JavaScript used to perfactly manage and handle events on multiple child elements
What is event bubbling in the DOM?
Event Bubbling is a work process that is happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
How do you create, add, and remove elements using JavaScript?
I can use document.createElement(), document.createTextNode(), appendChild(), element.remove() for create, add, and remove.
